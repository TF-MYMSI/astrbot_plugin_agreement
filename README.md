# 文档签订插件

## 功能说明

用户发送触发词后，机器人发送文档，用户回复「同意」或「不同意」确认，状态永久保存。

### 核心功能

- **协议签订**：触发词 → 发送文档 → 同意/拒绝 → 状态保存
- **拒绝静默**：拒绝后机器人不回复，但 `/doc_status` 和 `/doc_undo` 仍可用
- **反悔机制**：`/doc_undo` 可重新签订，支持次数和冷却限制
- **管理员特权**：管理员不受拒绝状态限制
- **私聊/群聊独立**：可分别控制
- **完全自定义**：触发词、同意词、拒绝词、文档内容、回复文案均可自定义

### 命令列表

| 命令 | 功能 | 权限 |
|------|------|------|
| `/doc_stats` | 查看统计 | 所有人 |
| `/doc_status` | 查看个人状态 | 所有人 |
| `/doc_undo` | 反悔重签 | 所有人 |
| `/doc_help` | 帮助 | 所有人 |
| `/doc_list` | 用户列表 | 管理员 |
| `/doc_reset` | 重置统计 | 管理员 |
| `/doc_reset_user <QQ号>` | 重置用户状态 | 管理员 |
| `/doc_reload` | 重载配置 | 管理员 |

### 配置方法

WebUI → 插件管理 → 文档签订插件 → 插件配置

所有配置项均为中文，按需修改即可。

### 安装方法

将 `astrbot_plugin_agreement` 放入 `/root/data/plugins/`，重载插件即可。

### 注意事项

- 拒绝后机器人静默，但 `/doc_status` 和 `/doc_undo` 仍可用
- 管理员不受拒绝状态限制
- 反悔有次数（默认3次）和冷却（默认24小时）限制
