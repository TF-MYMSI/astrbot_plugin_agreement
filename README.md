# 文档签订插件 (astrbot_plugin_agreement)

## 功能说明

用户发送指定关键词时，机器人自动发送文档，用户回复「同意」或「不同意」确认，状态永久保存。

本插件由 deepseek 辅助完成。

## 核心功能

| 功能 | 说明 |
|------|------|
| 首次触发发送 | 用户发送关键词时自动发送文档 |
| 同意/拒绝处理 | 记录用户选择，同意后正常聊天，拒绝后完全静默 |
| 管理员特权 | 管理员不受拒绝状态限制，可正常使用所有命令 |
| 冷却机制 | 防刷屏，可自定义冷却时间 |
| 私聊/群聊独立 | 可分别控制是否启用 |
| 触发词自定义 | 可任意增删触发关键词 |
| 同意/拒绝词自定义 | 可任意增删同意/拒绝关键词 |
| 统计功能 | 查看总用户数、同意率等 |
| 反悔机制 | 用户可自主反悔，重新签订协议 |
| 多章节文档 | 支持将协议拆分为多个章节 |

## 支持的命令

| 命令 | 功能 | 权限 |
|------|------|------|
| `/doc_stats` | 查看统计摘要 | 所有人（管理员不受拒限） |
| `/doc_status` | 查看个人状态 | 所有人（管理员不受拒限） |
| `/doc_help` | 显示帮助 | 所有人（管理员不受拒限） |
| `/doc_undo` | 反悔，重新签订协议 | 所有人（受次数/冷却限制） |
| `/doc_list` | 查看用户列表 | 管理员 |
| `/doc_reset` | 重置统计 | 管理员 |
| `/doc_reset_user <QQ号>` | 重置指定用户状态 | 管理员 |
| `/doc_reload` | 热重载配置 | 管理员 |

## 使用流程

1. 用户发送触发词（如「协议」）
2. 机器人发送文档
3. 用户回复「同意」或「不同意」（关键词可自定义）
4. 同意后正常聊天
5. 拒绝后机器人完全静默，不回复任何消息
6. 用户可通过 `/doc_undo` 反悔重新签订
7. 管理员可通过 `/doc_reset_user <QQ号>` 解除用户的拒绝状态

## 触发方式

- **私聊**：发送任意消息（需开启）
- **群聊**：@机器人 并发送任意消息（需开启）

## 反悔机制说明

用户可以通过 `/doc_undo` 命令反悔并重新签订协议：

- 支持配置是否允许反悔
- 支持配置最大反悔次数
- 支持配置反悔冷却时间（防止频繁操作）
- 管理员不受反悔限制

## 管理员特权说明

管理员用户**不受拒绝状态限制**：

- 即使管理员自己处于拒绝状态，仍然可以正常使用所有命令
- 管理员可以随时解除其他用户的拒绝状态
- 管理员不受反悔次数和冷却限制

## 配置方法

WebUI → 插件管理 → 文档签订插件 → 插件配置

### 基础配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `管理员列表` | 管理员QQ号列表 | `["123456", "789012"]` |
| `启用私聊` | 是否在私聊中启用 | `true` / `false` |
| `启用群聊` | 是否在群聊中启用 | `true` / `false` |

### 文档配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `文档名称` | 协议文档名称 | `用户协议` |
| `文档版本` | 协议版本号 | `v1.0` |
| `更新日期` | 协议更新日期 | `2026-04-10` |
| `联系方式` | 联系方式信息 | `QQ群 000000000` |
| `文档章节` | 协议章节列表（支持多章节） | 见下方示例 |

### 关键词配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `触发关键词` | 触发协议发送的关键词列表 | `["协议", "规则", "条例"]` |
| `同意关键词` | 表示同意的关键词列表 | `["同意", "agree", "YES", "是", "接受"]` |
| `拒绝关键词` | 表示拒绝的关键词列表 | `["不同意", "disagree", "NO", "否", "拒绝"]` |

### 回复文案配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `同意后回复` | 用户同意后的回复内容 | `✅ 已记录你的同意...` |
| `拒绝后回复` | 用户拒绝后的回复内容 | `❌ 已记录你的拒绝...` |
| `等待时回复` | 等待用户确认时的提示 | `📝 请回复「同意」或「不同意」...` |

### 反悔机制配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `允许反悔` | 是否允许用户反悔 | `true` / `false` |
| `最大反悔次数` | 用户最大反悔次数 | `3` |
| `反悔冷却时间` | 反悔冷却时间（秒） | `86400`（24小时） |

### 其他配置

| 配置项 | 说明 | 示例 |
|--------|------|------|
| `冷却时间` | 重复发送协议的冷却时间（秒） | `30` |
| `发送文字协议` | 是否发送文字版协议 | `true` / `false` |
