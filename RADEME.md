# 文档签订插件 (astrbot_plugin_agreement)

## 功能说明

用户发送指定关键词时，机器人自动发送文档（文字/图片），用户回复「同意」或「不同意」确认，状态永久保存。

### 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 首次触发发送 | 用户发送关键词时自动发送文档 |
| 同意/拒绝处理 | 记录用户选择，同意后正常聊天，拒绝后不回复 |
| 冷却机制 | 防刷屏，可自定义冷却时间 |
| 私聊/群聊独立 | 可分别控制是否启用 |
| 触发词自定义 | 可任意增删触发关键词 |
| 发送方式 | 支持文字协议、图片协议，可同时启用 |
| 统计功能 | 查看总用户数、同意率等 |
| 管理员命令 | 查看用户列表、重置统计、热重载配置 |

### 支持的命令

| 命令 | 功能 | 权限 |
| :--- | :--- | :--- |
| `/doc_stats` | 查看统计摘要 | 所有人 |
| `/doc_status` | 查看个人状态 | 所有人 |
| `/doc_help` | 显示帮助 | 所有人 |
| `/doc_list` | 查看用户列表 | 管理员 |
| `/doc_reset` | 重置统计 | 管理员 |
| `/doc_reload` | 热重载配置 | 管理员 |

### 使用流程

1. 用户发送触发词（如「协议」）
2. 机器人发送文档
3. 用户回复「同意」或「不同意」
4. 同意后正常聊天，拒绝后不再回复

### 配置方法

WebUI → 插件管理 → 文档签订插件 → 插件配置

可配置项：管理员列表、私聊/群聊开关、文档名称、触发关键词、冷却时间、发送方式、图片地址、回复内容、文档章节

### 图片配置

- 网络图片：填写图片URL
- 本地图片：填写服务器本地路径（如 `/root/data/protocol.png`）
- 图文同时：勾选「发送文字协议」和「发送图片协议」

### 安装方法

将 `astrbot_plugin_agreement` 文件夹放入 `/root/data/plugins/`，重载插件后启用。
